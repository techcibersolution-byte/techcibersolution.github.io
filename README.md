# Fondos “wallpaper” (HD) para el sitio

## Qué poner aquí

Coloca tu imagen de fondo HD de tecnología en esta carpeta con el nombre:

- `tech-wallpaper.jpg`

Rutas esperadas:

- `PROYECTO_INFORMATICO/assets/tech-wallpaper.jpg`

## Recomendaciones

- Formato: **JPG** o **WEBP**
- Resolución: **1920×1080** o superior (4K si quieres)
- Peso: idealmente **< 800 KB** (para que cargue rápido)

## Si usas otro nombre

Edita la línea en `styles.css`:

- `background-image: url("./assets/tech-wallpaper.jpg");`
